## 1. ความเป็นมา [TOR หน้า 1-2 ข้อ 1]

สำนักงานตำรวจแห่งชาติเป็นหน่วยงานบังคับใช้กฎหมายซึ่งต้องการระบบสารสนเทศที่บูรณาการการบริหารยุทธศาสตร์ การปฏิบัติงาน และข้อมูลในระดับองค์กร เพื่อยกระดับสำนักงานยุทธศาสตร์ตำรวจให้เป็น **Digital Strategic Hub**

ระบบ EOP จะรวบรวมข้อมูลข่าวสารและสถิติทุกแหล่ง ใช้ AI ในการประมวลผลและสนับสนุนการตัดสินใจ พัฒนา Big Data + Data Analytics + Data-Driven Decision Making

TOR กล่าวถึงมาตรฐานที่ยกระดับเทียบได้ เช่น Public Service Awards / WSIS Prizes

### 9.2.6 PoC (30 คะแนน) — 4 ฟีเจอร์ [TOR หน้า 50-56]

#### PoC 1: AI Command Drafting (5 คะแนน) [TOR หน้า 50-51]

- **Input:** 5 รายการ (หัวเรื่อง / หน่วยงาน / วัตถุประสงค์ / ข้อสั่งการ / ระยะเวลา)
- **Output:** ร่างหนังสือสั่งการที่มีลักษณะหนังสือราชการ
- **Scoring:**
  - ครบ 2 ด้าน (Format + Content) = **5 pt**
  - ครบ 1 ด้าน = 2.5 pt
  - ไม่ครบ = 0

#### PoC 2: Document Classification (10 คะแนน) [TOR หน้า 51-52]

- **Pre-test:** ตร. ให้ตัวอย่าง **16 ฉบับ**
- **วัน PoC:** ตร. ให้เอกสารใหม่ **16 ฉบับ** (DOCX)
- **6 หมวด:** ยศ. / ผบ. / มค. / มข. / วจ. / อจ.สยศ.ตร.

#### PoC 3: OCR ภาษาไทย (10 คะแนน) [TOR หน้า 52-53]

- **Input:** PDF 6 ไฟล์, ≥ 300 dpi, ตัวอักษร ≥ 10 point
- **CER:** `(S + D + I) / N × 100`

| CER | คะแนน |
|---|---|
| ≤ 10% | **10 pt** |
| 10-20% | 5 pt |
| 20-30% | 2.5 pt |
| > 30% | 0 pt |

#### PoC 4: XR Command Center (5 คะแนน) [TOR หน้า 53-56]

ต้องสาธิตด้วย XR Headset + prototype:
- Virtual Screens **60-62 จอ** (หรือดีกว่า)
- Dashboard 4 chart types: **Line / Bar / Pie / Map**
- GIS Heatmap / Bubble Map
- Dashboard Interaction ภายใน XR
- Virtual Screens 2+ พร้อมกัน
