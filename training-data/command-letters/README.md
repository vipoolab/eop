## 🎯 รูปแบบที่พบบ่อย — สำหรับ AI Training

### โครงสร้างหนังสือสั่งการ (Pattern)

หนังสือสั่งการตำรวจจริงมีรูปแบบ:

```
[ตราครุฑ]
คำสั่ง [หน่วยงาน]
ที่ [เลขที่]/[ปี พ.ศ.]
เรื่อง [หัวข้อ]
————————————————

[ความนำ — อ้างเหตุผล / กฎหมายอ้างอิง]
ด้วย / โดยที่ / ตามที่ ...

[เนื้อหา — ข้อสั่งการ]
อาศัยอำนาจตาม...
จึงให้ดำเนินการดังนี้
  ๑. ...
  ๒. ...
  ๓. ...

[ปิดท้าย]
ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป
สั่ง ณ วันที่ [วันที่] [เดือน] พ.ศ. [ปี]

[ลายเซ็น]
([ยศ-ชื่อ-นามสกุล])
[ตำแหน่ง]
```

### ลักษณะภาษาที่ใช้

- **คำเปิด**: "ด้วย", "โดยที่", "ตามที่", "เพื่อให้"
- **คำสั่ง**: "อาศัยอำนาจตาม...", "จึงให้...", "จึงสั่งการให้..."
- **คำเชื่อม**: "ทั้งนี้", "อนึ่ง", "ในการนี้"
- **คำปิด**: "ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป", "จึงเรียนมาเพื่อทราบและถือปฏิบัติโดยเคร่งครัด"
- **ระบบเลขข้อ**: ใช้เลขไทย ๑./๒./๓. (ไม่ใช่ 1./2./3.)
- **วันที่**: ใช้พุทธศักราช, รูปแบบ "วันที่ ๒๖ พฤศจิกายน พ.ศ. ๒๕๖๗"

### ระดับการสั่งการ

1. **ระดับ ตร.** (national) — ผบ.ตร. ลงนาม → ทุกหน่วยทั่วประเทศ
2. **ระดับ บช./ภ.** (regional) — ผบช./ภ. ลงนาม → หน่วยในเขตรับผิดชอบ
3. **ระดับ บก./ภ.จว.** (division) — ผบก./ภ.จว. ลงนาม → สถานีในสังกัด
4. **ระดับสถานี** (station) — ผกก. ลงนาม → ภายในสถานี

## ⚙️ การนำไปใช้ Train AI

### Use case 1: Few-shot prompting (PoC 1 — AI Command Drafter)

นำ 3–5 ตัวอย่างจากโฟลเดอร์นี้ใส่ใน system prompt ก่อนสั่งให้ AI ร่าง:

```
ตัวอย่างหนังสือสั่งการ ตร. (3 ฉบับแรกจาก /1-คำสั่ง-ตร):

[ใส่ extracted text ของแต่ละฉบับ]

ตอนนี้ ให้ร่างหนังสือสั่งการตามรูปแบบเดียวกัน โดยมี:
- หัวข้อ: [user input]
- ผู้รับ: [user input]
- ข้อสั่งการ: [user input]
```

### Use case 2: Fine-tuning (อนาคต)

แปลง PDF → text → JSON pairs (instruction, output) — สามารถใช้สำหรับ Claude fine-tuning หรือ LoRA training

### Use case 3: PoC 2 (Doc Classification)

ใช้เป็น labeled training data:
- คำสั่ง ตร. → label "ผบ." (กองแผนงานอำนวยการ)
- กฎ ก.ตร. → label "ยศ." (กองยุทธศาสตร์)
- แผนปฏิบัติราชการ → label "ยศ."
- มาตรการวินัย → label "อจ." (ฝ่ายอำนวยการ)

### Use case 4: PoC 3 (OCR Thai)

ไฟล์ขนาดใหญ่ที่เป็น scan (5-9 MB) เช่น "คำสั่ง-ตร-419-2556-การสอบสวน.pdf" เหมาะที่จะใช้ทดสอบ OCR เพราะเป็น scan ของเอกสารราชการจริง

## ⚠️ ข้อควรระวัง

1. เอกสารเหล่านี้เป็น **public domain** จากเว็บราชการ — สามารถใช้ training ภายในได้
2. **ห้ามนำไปเผยแพร่ต่อสาธารณะ** หากไม่จำเป็น — ให้เก็บไว้เพื่อใช้ภายในโครงการ
3. หากต้องการเอกสารเพิ่มเติม เช่น คำสั่งเฉพาะหน่วย หรือคำสั่งล่าสุด สามารถติดต่อหน่วยงานต้นเรื่องโดยตรง
4. ข้อมูล PII (ชื่อ ตำแหน่ง) ที่ปรากฏในเอกสารเหล่านี้เป็น **ข้อมูลสาธารณะ** จากเว็บราชการ — แต่ก็ควรระวังการนำไปใช้ในที่อื่น
